# Prompt Engineering Basics

This folder contains interactive Python demos for core prompt engineering patterns.

Each file is a standalone example:

- `zero_shot_prompting.py`
- `few_shot_prompting.py`
- `chain_of_thoughts.py`
- `role_prompting.py`
- `instruction_tuning.py`
- `prompt_chaining.py`
- `output_parsing.py`

## Run examples

Use Python to run each file separately, for example:

```bash
python prompt_engineering_basics/zero_shot_prompting.py
```

Each script asks for input and shows how the prompt is built and how a model might respond.
